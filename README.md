# 九龙拉棺Mod【一键上buff】(AddBuffMod)

这是一个简单的鸭科夫 Mod，允许你通过热键为自己添加正面的 Buff。  
创意工坊链接：https://steamcommunity.com/sharedfiles/filedetails/?id=3595021711

## 如何使用

在游戏中**长按**以下按键 **1 秒钟**以激活对应的 Buff：

* **[ 1 ]**: 激活 **[缓疗]**
* **[ 2 ]**: 激活 **[加速]**
* **[ 3 ]**: 激活 **[强翅]** (后坐力控制)
* **[ 4 ]**: 激活 **[负重]** (最大负重)
* **[ 5 ]**: 激活 **[护甲]**
* **[ 6 ]**: 激活 **[热血]** 
* **[ 7 ]**: 激活 **[镇静]** (长效止痛)
* **[ 8 ]**: 激活 **[明视]** (夜视)
* **[ 9 ]**: 激活 **[持久]** (耐力)

---

### 特殊按键

* **[ 0 ]**: 激活 **[九龙拉棺]**
    * **效果**: 一次性激活**上述全部 9 种 Buff**。
    * **那我问你**: "怎么感觉头顶尖尖的？"。
